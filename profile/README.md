## Overview

Yoink Advanced is a temporary file shelf that simplifies drag-and-drop operations by providing a persistent staging area where files can be parked between picking them up from a source and dropping them at a destination. It solves the fundamental drag-and-drop problem of needing to hold files while navigating to the destination — switching Spaces, scrolling to a different folder, or switching applications — by providing a shelf that holds the files until you are ready to complete the operation.

Drag shelf appears when a drag operation starts, providing a drop target that accepts files, text, images, and other draggable content. The shelf holds the dropped items until they are dragged out to their final destination, decoupling the pick-up and drop-off stages of drag-and-drop. Multiple items can be collected from different sources before being moved together.

Multi-source collection enables gathering files from multiple windows, folders, and applications before moving them to a single destination, handling the common scenario of collecting attachments from different locations for an email or gathering assets from different folders for a project.

Persistent shelf retains parked files across application switches, Space transitions, and window navigations, making the shelf available whenever the drag-and-drop operation resumes. The persistence handles the interruptions that naturally occur during multi-step file operations.

For users who frequently move files between locations and want drag-and-drop to work across windows, Spaces, and applications without holding the mouse button through navigation, Yoink Advanced is the file shelf that makes multi-step moves simple.
